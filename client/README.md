# Todo Client

## Running the app

1.  Create .env file by cloning the [.env.sample](/client/.env.sample)

```bash
$ cp .env.sample .env
```

2.  Fill the .env file with corresponding variables

3.  Install packages

```bash
$ yarn install
```

4. Run the app

```bash
$ yarn serve
```

## Running tests

1.  Create .env file by cloning the [.env.sample](/client/.env.sample)

```bash
$ cp .env.sample .env
```

2.  Fill the .env file with corresponding variables

3.  Run the tests

```bash
# unit tests
$ yarn test:unit

# unit tests
$ yarn test:e2e
```
